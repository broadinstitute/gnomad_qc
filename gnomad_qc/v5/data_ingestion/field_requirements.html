This table provides a view of each global (non-row) field. Every leaf field (i.e., non-struct field) is listed on its own row with the full annotation. adj definition: A variant is considered to fail the "adj" criteria if any of the following conditions are met: GQ < 20, DP < 10, or AB < 0.2 (heterozygous calls).
