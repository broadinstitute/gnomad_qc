gnomad_qc.v5.resources.sample_qc.get_gen_anc_ht(version='5.0', test=False, data_type='joint', projection_only=False)[source]

Get the TableResource of samples’ inferred genetic ancestry group for the indicated gnomAD version.

Parameters:

  • version (str) – Version of gen anc group TableResource to return.

  • test (bool) – Whether to use the test version of the genetic ancestry TableResource.

  • data_type (str) – Data type used in sample QC, e.g. “exomes” or “joint”.

  • projection_only (bool) – Whether the inference results consist of just the results for the projected samples. When set to False, probability scores will not be included as they cannot be obtained for all samples.
