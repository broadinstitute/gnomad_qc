gnomad_qc.v4.sample_qc.hard_filters

Script to determine samples that fail hard filtering thresholds.

usage: gnomad_qc.v4.sample_qc.hard_filters.py [-h] [--overwrite] [--test]
                                              [--sample-qc]
                                              [--n-alt-alleles-strata N_ALT_ALLELES_STRATA]
                                              [--n-alt-alleles-strata-name N_ALT_ALLELES_STRATA_NAME]
                                              [--sample-qc-n-partitions SAMPLE_QC_N_PARTITIONS]
                                              [--compute-coverage]
                                              [--compute-contamination-estimate]
                                              [--contam-dp-cutoff CONTAM_DP_CUTOFF]
                                              [--calling-interval-name {ukb,broad,intersection}]
                                              [--calling-interval-padding {0,50}]
                                              [--compute-chr20-mean-dp]
                                              [--compute-qc-mt-callrate]
                                              [--qc-mt-callrate-min-af QC_MT_CALLRATE_MIN_AF]
                                              [--qc-mt-callrate-min-site-callrate QC_MT_CALLRATE_MIN_SITE_CALLRATE]
                                              [--compute-hard-filters]
                                              [--include-sex-filter]
                                              [--max-n-singleton MAX_N_SINGLETON]
                                              [--max-r-het-hom-var MAX_R_HET_HOM_VAR]
                                              [--min-bases-dp-over-1 MIN_BASES_DP_OVER_1]
                                              [--min-bases-dp-over-20 MIN_BASES_DP_OVER_20]
                                              [--max-chimera MAX_CHIMERA]
                                              [--max-contamination-estimate MAX_CONTAMINATION_ESTIMATE]
                                              [--min-cov MIN_COV]
                                              [--min-qc-mt-adj-callrate MIN_QC_MT_ADJ_CALLRATE]
                                              [--slack-channel SLACK_CHANNEL]

Named Arguments

--overwrite

Overwrite all Matrixtables/Tables. (default: False).

Default: False

--test

Use the v4 test dataset instead of the full dataset.

Default: False

--sample-qc

Compute Hail’s VDS sample QC metrics.

Default: False

--n-alt-alleles-strata

Number of alternative alleles to stratify sample QC on. For example ‘n_alt_alleles_strata’ = 2 will stratify ‘bi-allelic’ and ‘multi-allelic’. ‘n_alt_alleles_strata’ = 3 will stratify under 3 alt alleles and 3 or more alt alleles. Default is 3.

Default: 3

--n-alt-alleles-strata-name

Name to use for number of alternative allele stratification, where strata names are ‘under_{n_alt_alleles_strata_name}_alt_alleles’ and ‘{n_alt_alleles_strata_name}_or_more_alt_alleles’. This is not used when n_alt_alleles_strata = 2, and instead uses ‘bi-allelic’ and ‘multi-allelic’. Default is ‘three’.

Default: “three”

--sample-qc-n-partitions

Number of desired partitions for the sample QC output Table.

Default: 500

--compute-coverage

Compute per interval coverage metrics using Hail’s vds.interval_coverage method.

Default: False

--compute-contamination-estimate

Compute contamination estimate as the mean of reference allele balances of high-quality (DP >= contam-dp-cutoff), autosomal, bi-allelic homozygous SNVs per sample.

Default: False

--contam-dp-cutoff

Minimum genotype depth to be included in contamination estimate calculation.

Default: 10

--calling-interval-name

Possible choices: ukb, broad, intersection

Name of calling intervals to use for interval coverage. One of: ‘ukb’, ‘broad’, or ‘intersection’.

Default: “intersection”

--calling-interval-padding

Possible choices: 0, 50

Number of base pair padding to use on the calling intervals. One of 0 or 50 bp.

Default: 50

--compute-chr20-mean-dp

Compute per sample mean DP on chromosome 20 using interval coverage results.

Default: False

--compute-qc-mt-callrate

Compute per sample callrate on the predetermined QC variants after filtering to common variants (AF > –qc-mt-callrate-min-af) with high site callrate ( > –qc-mt-callrate-min-site-callrate) for ADJ genotypes.

Default: False

--qc-mt-callrate-min-af

Filtering threshold to use for minimum variant allele frequency when computing sample callrate on the predetermined QC variants (–compute-qc-mt-callrate). Default is 0.0001.

Default: 0.0001

--qc-mt-callrate-min-site-callrate

Filtering threshold to use for minimum site ADJ callrate when computing sample callrate on predetermined QC variants (–compute-qc-mt-callrate). Default is 0.99.

Default: 0.99

--compute-hard-filters

Computes samples to be hard-filtered. NOTE: Cutoffs should be determined by visual inspection of the metrics.

Default: False

--include-sex-filter

If sex filters should be included in hard filtering.

Default: False

--slack-channel

Slack channel to post results and notifications to.

Hard-filter cutoffs

Arguments used for hard-filter cutoffs.

--max-n-singleton

Filtering threshold to use for the maximum number of singletons. Default is 5000.

Default: 5000

--max-r-het-hom-var

Filtering threshold to use for the maximum ratio of heterozygotes to alternate homozygotes. Default is 10.

Default: 10

--min-bases-dp-over-1

Filtering threshold to use for the minimum number of bases with a DP over one. Default is 5e7.

Default: 50000000.0

--min-bases-dp-over-20

Filtering threshold to use for the minimum number of bases with a DP over 20. Default is 4e7.

Default: 40000000.0

--max-chimera

Filtering threshold to use for maximum chimera (this is a proportion not a percent, e.g. 5% == 0.05, %5 != 5). Default is 0.05.

Default: 0.05

--max-contamination-estimate

Filtering threshold to use for maximum contamination estimate (from –compute-contamination-estimate). Default is 0.015.

Default: 0.015

--min-cov

Minimum chromosome 20 coverage for inclusion when computing hard-filters.

--min-qc-mt-adj-callrate

Minimum sample callrate computed on only predetermined QC variants (predetermined using CCDG genomes/exomes, gnomAD v3.1 genomes, and UKB exomes) after ADJ filtering.

  • n_alt_alleles_strata_name (str) – Name to use for number of alternative allele stratification, where strata names are ‘under_{n_alt_alleles_strata_name}_alt_alleles’ and ‘{n_alt_alleles_strata_name}_or_more_alt_alleles’. This is not used when n_alt_alleles_strata = 2, and instead uses ‘bi-allelic’ and ‘multi-allelic’. Default is ‘three’.

Return type:

Table

Returns:

Table containing sample QC metrics.

gnomad_qc.v4.sample_qc.hard_filters.compute_hard_filters(include_sex_filter=False, max_n_singleton=5000, max_r_het_hom_var=10, min_bases_dp_over_1=50000000.0, min_bases_dp_over_20=40000000.0, max_chimera=0.05, max_contamination_estimate=0.015, test=False, chr20_mean_dp_ht=None, min_cov=None, qc_mt_callrate_ht=None, min_qc_mt_adj_callrate=None)[source]

Apply hard filters to samples and return a Table with the filtered samples and the reason for filtering.

If include_sex_filter is True, this function expects a sex inference Table generated by sex_inference.py –impute-sex.

Warning

The defaults used in this function are callset specific, these hardfilter cutoffs will need to be re-examined for each callset

Parameters:

  • include_sex_filter (bool) – If sex inference should be used in filtering.

  • max_n_singleton (float) – Filtering threshold to use for the maximum number of singletons.

  • max_r_het_hom_var (float) – Filtering threshold to use for the maximum ratio of heterozygotes to alternate homozygotes.

  • min_bases_dp_over_1 (float) – Filtering threshold to use for the minimum number of bases with a DP over one.

  • min_bases_dp_over_20 (float) – Filtering threshold to use for the minimum number of bases with a DP over 20.

  • max_chimera (float) – Filtering threshold to use for maximum chimera (this is a proportion not a percent, e.g. 5% == 0.05, %5 != 5).

  • max_contamination_estimate (float) – Filtering threshold to use for maximum contamination estimate.

  • test (bool) – Whether to use the gnomAD v4 test dataset. Default is to use the full dataset.

  • chr20_mean_dp_ht (Table) – Table containing the per sample chromosome 20 mean DP.

  • min_cov (int) – Filtering threshold to use for chr20 coverage.

  • qc_mt_callrate_ht (Table) – Table containing the per sample QC MatrixTable callrate.

  • min_qc_mt_adj_callrate (float) – Filtering threshold to use for sample callrate computed on only predetermined QC variants (predetermined using CCDG genomes/exomes, gnomAD v3.1 genomes, and UKB exomes) after ADJ filtering.

Return type:

Table

Returns:

Table of hard filtered samples.
