gnomad_qc.v4.sample_qc.hard_filtersÔÉÅ

Script to determine samples that fail hard filtering thresholds.

usage: gnomad_qc.v4.sample_qc.hard_filters.py [-h] [--overwrite] [--test]

                                              [--n-alt-alleles-strata N_ALT_ALLELES_STRATA]

                                              [--compute-coverage]
                                              [--compute-contamination-estimate]

Named ArgumentsÔÉÅ

--overwrite

Overwrite all Matrixtables/Tables. (default: False).

Default: False

--test

Use the v4 test dataset instead of the full dataset.

Default: False

--sample-qc

Compute Hail‚Äôs VDS sample QC metrics.

Default: False

--n-alt-alleles-strata

Number of alternative alleles to stratify sample QC on. For example ‚Äòn_alt_alleles_strata‚Äô = 2 will stratify ‚Äòbi-allelic‚Äô and ‚Äòmulti-allelic‚Äô. ‚Äòn_alt_alleles_strata‚Äô = 3 will stratify under 3 alt alleles and 3 or more alt alleles. Default is 3.

Default: 3

--n-alt-alleles-strata-name

Name to use for number of alternative allele stratification, where strata names are ‚Äòunder_{n_alt_alleles_strata_name}_alt_alleles‚Äô and ‚Äò{n_alt_alleles_strata_name}_or_more_alt_alleles‚Äô. This is not used when n_alt_alleles_strata = 2, and instead uses ‚Äòbi-allelic‚Äô and ‚Äòmulti-allelic‚Äô. Default is ‚Äòthree‚Äô.

Default: ‚Äúthree‚Äù

--sample-qc-n-partitions

Number of desired partitions for the sample QC output Table.

Default: 500

--compute-coverage

Compute per interval coverage metrics using Hail‚Äôs vds.interval_coverage method.

Default: False

--compute-contamination-estimate

Compute contamination estimate as the mean of reference allele balances of high-quality (DP >= contam-dp-cutoff), autosomal, bi-allelic homozygous SNVs per sample.

Default: False

--contam-dp-cutoff

Minimum genotype depth to be included in contamination estimate calculation.

Default: 10

--calling-interval-name

Possible choices: ukb, broad, intersection

Name of calling intervals to use for interval coverage. One of: ‚Äòukb‚Äô, ‚Äòbroad‚Äô, or ‚Äòintersection‚Äô.

Default: ‚Äúintersection‚Äù

--calling-interval-padding

Possible choices: 0, 50

Number of base pair padding to use on the calling intervals. One of 0 or 50 bp.

Default: 50

--compute-chr20-mean-dp

Compute per sample mean DP on chromosome 20 using interval coverage results.

Default: False

--compute-qc-mt-callrate

Compute per sample callrate on the predetermined QC variants after filtering to common variants (AF > ‚Äìqc-mt-callrate-min-af) with high site callrate ( > ‚Äìqc-mt-callrate-min-site-callrate) for ADJ genotypes.

Default: False

--qc-mt-callrate-min-af

Filtering threshold to use for minimum variant allele frequency when computing sample callrate on the predetermined QC variants (‚Äìcompute-qc-mt-callrate). Default is 0.0001.

Default: 0.0001

--qc-mt-callrate-min-site-callrate

Filtering threshold to use for minimum site ADJ callrate when computing sample callrate on predetermined QC variants (‚Äìcompute-qc-mt-callrate). Default is 0.99.

Default: 0.99

--compute-hard-filters

Computes samples to be hard-filtered. NOTE: Cutoffs should be determined by visual inspection of the metrics.

Default: False

--include-sex-filter

If sex filters should be included in hard filtering.

Default: False

--slack-channel

Slack channel to post results and notifications to.

Hard-filter cutoffsÔÉÅ

Arguments used for hard-filter cutoffs.

--max-n-singleton

Filtering threshold to use for the maximum number of singletons. Default is 5000.

Default: 5000

--max-r-het-hom-var

Filtering threshold to use for the maximum ratio of heterozygotes to alternate homozygotes. Default is 10.

Default: 10

--min-bases-dp-over-1

Filtering threshold to use for the minimum number of bases with a DP over one. Default is 5e7.

Default: 50000000.0

--min-bases-dp-over-20

Filtering threshold to use for the minimum number of bases with a DP over 20. Default is 4e7.

Default: 40000000.0

--max-chimera

Filtering threshold to use for maximum chimera (this is a proportion not a percent, e.g. 5% == 0.05, %5 != 5). Default is 0.05.

Default: 0.05

--max-contamination-estimate

Filtering threshold to use for maximum contamination estimate (from ‚Äìcompute-contamination-estimate). Default is 0.015.

Default: 0.015

--min-cov

Minimum chromosome 20 coverage for inclusion when computing hard-filters.

--min-qc-mt-adj-callrate

Minimum sample callrate computed on only predetermined QC variants (predetermined using CCDG genomes/exomes, gnomAD v3.1 genomes, and UKB exomes) after ADJ filtering.

Apply hard filters to samples and return a Table with the filtered samples and the reason for filtering.

If include_sex_filter is True, this function expects a sex inference Table generated by sex_inference.py ‚Äìimpute-sex.

Warning

The defaults used in this function are callset specific, these hardfilter cutoffs will need to be re-examined for each callset

Parameters:

  • max_n_singleton (float) ‚Äì Filtering threshold to use for the maximum number of singletons.

  • max_r_het_hom_var (float) ‚Äì Filtering threshold to use for the maximum ratio of heterozygotes to alternate homozygotes.

  • min_bases_dp_over_1 (float) ‚Äì Filtering threshold to use for the minimum number of bases with a DP over one.

  • min_bases_dp_over_20 (float) ‚Äì Filtering threshold to use for the minimum number of bases with a DP over 20.

  • max_chimera (float) ‚Äì Filtering threshold to use for maximum chimera (this is a proportion not a percent, e.g. 5% == 0.05, %5 != 5).

  • max_contamination_estimate (float) ‚Äì Filtering threshold to use for maximum contamination estimate.

  • test (bool) ‚Äì Whether to use the gnomAD v4 test dataset. Default is to use the full dataset.

  • chr20_mean_dp_ht (Table) ‚Äì Table containing the per sample chromosome 20 mean DP.

  • min_cov (int) ‚Äì Filtering threshold to use for chr20 coverage.

  • qc_mt_callrate_ht (Table) ‚Äì Table containing the per sample QC MatrixTable callrate.

  • min_qc_mt_adj_callrate (float) ‚Äì Filtering threshold to use for sample callrate computed on only predetermined QC variants (predetermined using CCDG genomes/exomes, gnomAD v3.1 genomes, and UKB exomes) after ADJ filtering.

Return type:

Table

Returns:
